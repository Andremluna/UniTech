# UniTech 🚀

Landing page institucional da **UniTech**, empresa de desenvolvimento de software focada em tecnologia ágil e alta performance.

## 🌐 Acesse o site

👉 [andremluna.github.io/UniTech](https://andremluna.github.io/UniTech)

---

## 📋 Sobre o Projeto

A UniTech é uma empresa que desenvolve sistemas web e aplicativos mobile sob medida, utilizando metodologias ágeis para entregar valor de forma contínua e transparente aos clientes.

Esta landing page foi desenvolvida a partir de um design criado no **Figma**, recriado fielmente em **HTML, CSS e JavaScript puro**, sem frameworks ou dependências externas.

---

## 🗂️ Seções da Página

- **Hero** — Apresentação da empresa com chamada para ação
- **Valores** — O que diferencia uma página que converte, com timeline animada
- **Metodologias Ágeis** — Scrum e Kanban explicados com detalhes
- **Serviços** — Sistemas Web e Apps Mobile
- **Time** — Equipe responsável pelo projeto
- **FAQ** — Perguntas frequentes
- **Rodapé** — Informações de contato

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica
- **CSS3** — Estilização, animações e responsividade
- **JavaScript** — Interatividade (FAQ accordion, scroll animations, navbar)
- **Google Fonts** — Poppins, Manrope, Space Grotesk

---

## ✨ Funcionalidades

- ✅ Design responsivo (mobile e desktop)
- ✅ Animações suaves no scroll (reveal on scroll)
- ✅ Timeline animada na seção de valores
- ✅ Navbar fixa com efeito de blur ao rolar
- ✅ FAQ com accordion interativo
- ✅ Navegação suave entre seções (smooth scroll)
